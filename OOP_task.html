3. Output of Statements

Question: Show the output of the following statements:

a. System.out.println("1" + 1);

b. System.out.println('1' + 1);

c. System.out.println("1" + 1 + 1);

d. System.out.println("1" + (1 + 1));

e. System.out.println('1' + 1 + 1);

Answer:

a. System.out.println("1" + 1); → 11

// String concatenation, not addition

b. System.out.println('1' + 1); → 50

// '1' is ASCII 49, so 49 + 1 = 50

c. System.out.println("1" + 1 + 1); → 111

// "1" + 1 = "11", then "11" + 1 = "111"

d. System.out.println("1" + (1 + 1)); → 12

// Parentheses first: 1+1=2, then "1"+2="12"

e. System.out.println('1' + 1 + 1); → 51

// '1'+1=50, then 50+1=51

2. Implementing Two Methods

Question: Create your java code to implement any two methods from question 1 as static methods, examine your code, from the same class.

Answer:

public class MathUtils {
    
    // calculating hypotenuse using Pythagoras
    public static double hypotenuse(double side1, double side2) {
        return Math.sqrt(side1 * side1 + side2 * side2);
    }
    
    // finding smallest of three numbers
    public static int smallest(int x, int y, int z) {
        int min = x;
        if (y < min) {
            min = y;
        }
        if (z < min) {
            min = z;
        }
        return min;
    }
    
    public static void main(String[] args) {
        // testing hypotenuse
        double h = hypotenuse(3.0, 4.0);
        System.out.println("Hypotenuse: " + h);
        
        // testing smallest
        int small = smallest(5, 2, 8);
        System.out.println("Smallest: " + small);
    }
}

3. Employee Class

Question: Create a class called Employee. Class Employee includes three pieces of information as public instance variables: a first name (type String), a last name (type String) and a monthly salary (double). Your class should have a constructor that creates an employee with specified first name and last name. Provide a method setSalary(double salary) to set the monthly salary of the employee; if the monthly salary is not positive, set it to 0.0.

Write a test application named EmployeeTest that creates an employee object with first name "Ahmed", last name "Aly" and a salary equals to 1000, display the object's salary, then give the employee a 10% raise and display the salary again.

Answer:

public class Employee {
    public String firstName;
    public String lastName;
    public double monthlySalary;
    
    public Employee(String firstName, String lastName) {
        this.firstName = firstName;
        this.lastName = lastName;
    }
    
    public void setSalary(double salary) {
        if (salary > 0) {
            this.monthlySalary = salary;
        } else {
            this.monthlySalary = 0.0;
        }
    }
}

public class EmployeeTest {
    public static void main(String[] args) {
        Employee emp = new Employee("Ahmed", "Aly");
        emp.setSalary(1000);
        
        System.out.println("Initial salary: " + emp.monthlySalary);
        
        // 10% raise
        emp.setSalary(emp.monthlySalary * 1.10);
        System.out.println("After raise: " + emp.monthlySalary);
    }
}

4. Stock Class

Question: Design a class named Stock that contains:

  • A string data field named symbol for the stock's symbol.
  • A string data field named name for the stock's name.
  • A double data field named previousClosingPrice that stores the stock price for the previous day.
  • A double data field named currentPrice that stores the stock price for the current time.
  • A constructor that creates a stock with specified symbol and name.
  • The accessor methods for all data fields.
  • The mutator methods for previousClosingPrice and currentPrice.
  • A method named changePercent() that returns the percentage changed from previousClosingPrice to currentPrice.

Implement the class. Write a test program that creates a Stock object with the stock symbol SUNW, the name Sun Microsystems Inc, and the previous closing price of 100. Set a new current price to 90 and display the price-change percentage.

Answer:

public class Stock {
    private String symbol;
    private String name;
    private double previousClosingPrice;
    private double currentPrice;
    
    public Stock(String symbol, String name) {
        this.symbol = symbol;
        this.name = name;
    }
    
    // getters
    public String getSymbol() { 
        return symbol; 
    }
    
    public String getName() { 
        return name; 
    }
    
    public double getPreviousClosingPrice() { 
        return previousClosingPrice; 
    }
    
    public double getCurrentPrice() { 
        return currentPrice; 
    }
    
    // setters
    public void setPreviousClosingPrice(double price) {
        this.previousClosingPrice = price;
    }
    
    public void setCurrentPrice(double price) {
        this.currentPrice = price;
    }
    
    // calculating change percentage
    public double changePercent() {
        return ((currentPrice - previousClosingPrice) / previousClosingPrice) * 100;
    }
}

public class StockTest {
    public static void main(String[] args) {
        Stock stock = new Stock("SUNW", "Sun Microsystems Inc");
        stock.setPreviousClosingPrice(100);
        stock.setCurrentPrice(90);
        
        System.out.println("Price change: " + stock.changePercent() + "%");
    }
}

5. Smart Air Conditioner

Question: Create a class named AirConditioner that models a real-world AC unit.

1. Attributes (Private):

  • brand (String)
  • temperature (int)
  • isOn (boolean)

2. Constructors:

  • A constructor that takes brand as a parameter. It should set the default temperature to 24 and isOn to false.

3. Methods:

  • turnOn(): Sets isOn to true.
  • turnOff(): Sets isOn to false.
  • setTemperature(int temp): Only changes the temperature if the AC is currently on and the new temperature is between 16 and 30. If the AC is off or the value is invalid, print an error message.
  • displayStatus(): Prints the brand, current state (On/Off), and temperature.

Answer:

public class AirConditioner {
    private String brand;
    private int temperature;
    private boolean isOn;
    
    public AirConditioner(String brand) {
        this.brand = brand;
        this.temperature = 24;  // default temp
        this.isOn = false;
    }
    
    public void turnOn() {
        isOn = true;
    }
    
    public void turnOff() {
        isOn = false;
    }
    
    public void setTemperature(int temp) {
        if (!isOn) {
            System.out.println("Error: AC is off! Turn it on first.");
        } else if (temp < 16 || temp > 30) {
            System.out.println("Error: Temp must be between 16 and 30.");
        } else {
            temperature = temp;
        }
    }
    
    public void displayStatus() {
        System.out.println("Brand: " + brand);
        System.out.println("Status: " + (isOn ? "On" : "Off"));
        System.out.println("Temperature: " + temperature);
    }
}

6. Library Book System

Question: Create a system to track books and the total number of books created.

1. Class Book:

  • Instance Variables: title (String), price (double).
  • Static Variable: totalBooks (int) -- initialized to 0.
  • Constructor: Accepts title and price. It must increment totalBooks every time a new object is created.
  • Static Method: displayTotalBooks() that prints the total number of books created so far.

2. Class Library:

  • Create a method addDiscount(Book b, double percentage) that accepts a Book object and reduces its price by the given percentage.

3. Main Class:

  • Create 3 distinct Book objects.
  • Call addDiscount on one of them.
  • Call displayTotalBooks() to prove the static counter works.

Answer:

public class Book {
    private String title;
    private double price;
    private static int totalBooks = 0;  // shared counter
    
    public Book(String title, double price) {
        this.title = title;
        this.price = price;
        totalBooks++;  // increment every time we create a book
    }
    
    public double getPrice() { 
        return price; 
    }
    
    public void setPrice(double price) { 
        this.price = price; 
    }
    
    public static void displayTotalBooks() {
        System.out.println("Total books: " + totalBooks);
    }
}

public class Library {
    public void addDiscount(Book b, double percentage) {
        double newPrice = b.getPrice() * (1 - percentage / 100);
        b.setPrice(newPrice);
    }
}

public class LibraryTest {
    public static void main(String[] args) {
        // creating 3 books
        Book book1 = new Book("Java Programming", 50.0);
        Book book2 = new Book("Data Structures", 60.0);
        Book book3 = new Book("Algorithms", 55.0);
        
        Library library = new Library();
        library.addDiscount(book1, 10);  // 10% off
        
        Book.displayTotalBooks();
        System.out.println("Book1 after discount: " + book1.getPrice());
    }
}

7. Employee Inheritance

Question: Note: Do not use polymorphism (e.g., do not use an Employee reference to hold a Manager object).

1. Class Employee:

  • Attributes: name, id.
  • Constructor: Accepts name and id.
  • Method: printDetails() displays the name and ID.

2. Class Manager (extends Employee):

  • Attributes: Adds department (String).
  • Constructor: Accepts name, id, and department. You must use the super keyword to initialize the inherited fields.
  • Method: printDetails() -- Override this method to print the Name, ID, and Department.

3. Main Class:

  • Create a Manager object (e.g., Manager m = new Manager(...)).
  • Call m.printDetails().

Answer:

public class Employee {
    private String name;
    private int id;
    
    public Employee(String name, int id) {
        this.name = name;
        this.id = id;
    }
    
    public void printDetails() {
        System.out.println("Name: " + name);
        System.out.println("ID: " + id);
    }
}

public class Manager extends Employee {
    private String department;
    
    public Manager(String name, int id, String department) {
        super(name, id);  // calling parent constructor
        this.department = department;
    }
    
    @Override
    public void printDetails() {
        super.printDetails();  // calling parent method
        System.out.println("Department: " + department);
    }
}

public class EmployeeTest {
    public static void main(String[] args) {
        Manager m = new Manager("John Smith", 1001, "Engineering");
        m.printDetails();
    }
}

1. Box Constructor Error

Question: Identify the error in the following code. Does it fail to compile?

class Box {
    double width, height;
    
    // Parameterized constructor
    public Box(double w, double h) {
        width = w;
        height = h;
    }
}

public class TestBox {
    public static void main(String[] args) {
        // Error here
        Box myBox = new Box();
        System.out.println(myBox.width);
    }

Answer:

Problem: Line 14 tries new Box() but there's no constructor that takes no arguments.

Solution 1 - Add no-arg constructor:

public Box() {
    width = 0;
    height = 0;
}

Solution 2 - Give it arguments:

Box myBox = new Box(5.0, 10.0);

2. Missing 'this' Keyword

Question: The following code compiles, but the output is incorrect (it prints 0 instead of 10). Fix the code inside the method to ensure the instance variable x is updated properly.

class Demo {
    int x;
    
    public void setX(int x) {
        x = x; // Logic Error Here
    }
    
    public void display() {
        System.out.println("x = " + x);
    }
}

public class Main {
    public static void main(String[] args) {
        Demo d = new Demo();
        d.setX(10);
        d.display();
    }
}

Answer:

Problem: x = x; just assigns the parameter to itself, doesn't update the instance variable.

Fix:

public void setX(int x) {
    this.x = x;  // 'this' refers to the object's x
}

3. Static/Non-static Issues

Question: Identify and fix the errors in the following code:

class Calculator {
    int count = 0; // Instance variable
    
    public static void increment() {
        count++; // Error 1
    }
    
    public void display() {
        System.out.println(count);
    }
}

public class Test {
    public static void main(String[] args) {
        Calculator.display(); // Error 2
    }
}

Answer:

Problem 1: Static method can't access instance variable count

Problem 2: Calling instance method display() on the class instead of object

Fix:

class Calculator {
    static int count = 0;  // make it static
    
    public static void increment() {
        count++;  // now it works
    }
    
    public static void display() {  // make this static too
        System.out.println(count);
    }
}

public class Test {
    public static void main(String[] args) {
        Calculator.increment();
        Calculator.display();  // now both work
    }
}

Q4. Tracing

1. Car Class Trace

Question: What is the output of the following code:

class Car {
    public double speed;
    public String model;
    public double price = 1000;
    public static double sales;
    
    public Car(String model, int speed) {
        this.speed = speed;
        this.model = model;
        sales = sales + price;
    }
    
    public void increaseSpeed(double value) {
        speed = speed + value;
    }
}

class Test {
    public static void main(String[] arg) {
        Car car1 = new Car("Opel Corsa", 20);
        Car car2 = new Car("Matrix", 20);
        car1.increaseSpeed(120);
        car2.increaseSpeed(100);
        System.out.println(car1.speed);
        System.out.println(car2.speed);
        System.out.println(car1.sales);
        System.out.println(car2.sales);
    }
}

Answer:

Output:

140.0
120.0
2000.0
2000.0

How it works:

  • car1 created: speed=20, sales becomes 1000
  • car2 created: speed=20, sales becomes 2000 (it's static!)
  • car1.increaseSpeed(120): car1.speed = 20 + 120 = 140
  • car2.increaseSpeed(100): car2.speed = 20 + 100 = 120
  • sales is static so both objects share it = 2000

2. Array Method Trace

Question: Show the printout of the following code:

public static void main(String[] arg) {
    int[] list = {1, 2, 3, 4, 5};
    m(list);
    for(int i = 0; i < list.length; i++)
        System.out.println(list[i]);
}

public static void m(int[] arr) {
    for(int i = 0; i < arr.length; i++)
        arr[i] = arr[i] + 5;
}

Answer:

Output:

6
7
8
9
10

Why:

  • Arrays are passed by reference (not copied)
  • Method m() adds 5 to each element
  • {1,2,3,4,5} becomes {6,7,8,9,10}
  • Changes stick because we modified the actual array

3. Static Variable Trace

Question: Show the printout of the following code:

class A {
    int i;
    static int j;
    
    A(int i) {
        this.i = i;
        j++;
    }
    
    void m1(A a1) {
        a1.i++;
        a1.j++;
    }
}

class B {
    public static void main(String[] arg) {
        A a1 = new A(4);
        A a2 = new A(7);
        a1.m1(a1);
        System.out.println("a1:" + "i=" + a1.i + " j=" + a1.j);
        System.out.println("a2:" + "i=" + a2.i + " j=" + a2.j);
    }
}

Answer:

Output:

a1:i=5 j=3
a2:i=7 j=3

Step by step:

  1. a1 = new A(4): a1.i=4, j=1 (static)
  2. a2 = new A(7): a2.i=7, j=2 (static, so it's now 2 for everyone)
  3. a1.m1(a1): increments a1.i to 5 and j to 3
  4. j is static, so both a1 and a2 see j=3
  5. a2.i wasn't touched, still 7
